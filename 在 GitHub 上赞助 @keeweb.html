keeweb
阿姆斯特丹

KeeWeb 是与 KeePass 兼容的免费开源密码管理器。该项目的目标是拥有一个设计精美的免费密码管理器，无需安装任何东西即可在您的浏览器中正常工作。﻿

免费开放

KeeWeb的自由度可以从不同的角度看出：

  • 该应用程序是免费的（免费），并且将始终保持免费，至少供个人使用；
  • 以及它现在和将来都是无广告的；
  • KeeWeb中使用的 KDBX 格式是许多应用程序都认可的众所周知的格式，绝对没有供应商锁定；
  • 你不需要安装任何东西来运行 KeeWeb，只要去app.keeweb.info；
  • 它建立在网络技术之上，这使得使用自定义 CSS 和 JavaScript 修改应用程序成为可能，KeeWeb 为此提供了插件 API；
  • 源代码在GitHub上可用，查看它以了解密码管理器内部发生的情况；
  • KeeWeb 可以通过多种方式轻松自托管；
  • 最后但同样重要的是，代码是在自由的MIT许可下提供的，因此其他开发人员可以基于 KeeWeb 中使用的技术构建他们的应用程序。

钱都花到哪儿了？

KeeWeb 是一个爱好项目，没有组织或个人全职工作。没有计划在应用程序中提供任何类型的付费订阅或烦人的广告。但是也有一些费用，例如：

  • 适用于 Windows 和 macOS 的代码签名证书；
  • 网站托管；
  • 域名;
  • 用于在不同平台上测试应用程序的资源；
  • 用于创建 KeeWeb 的软件许可证；
  • 用于存储私钥的硬件令牌；
  • 研究市场上可用的替代品；
  • 花在回答问题、提供支持、调试、审查代码等上的时间。

此外，每笔捐款都有助于保持动力，使项目继续存在。

目标

短期目标：

  • 支付建设项目和保持最新的所有成本；
  • 拥有一个有效的 Apple Developer 帐户来更新证书；
  • 升级托管解决方案；
  • 为项目分配更多时间。

中期目标：

  • 进行外部安全审计；
  • 开发适用于 iOS 和 Android 的应用程序。

长期目标：

  • 有专职维护人员；
  • 构建企业解决方案。
